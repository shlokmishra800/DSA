# Concept of Constructors in OOPs


## Constructors
- **Definition**: A constructor is a special method in a class that is automatically called when an object is created.
- **Explanation**: It initializes the object’s fields. Constructors have the same name as the class and do not have a return type.

### Types of Constructors

#### 1. Non-Parameterized Constructor
- **Definition**: A constructor that does not take any arguments.
- **Explanation**: It initializes objects with default values.

#### 2. Parameterized Constructor
- **Definition**: A constructor that takes arguments.
- **Explanation**: It allows initialization of objects with specific values at the time of creation.

#### 3. Copy Constructor
- **Definition**: A constructor that creates a new object as a copy of an existing object.
- **Explanation**: It takes another object of the same class as a parameter and copies its values into the new object.
