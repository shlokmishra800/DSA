# Concept of  Classes,Objects and Methods  is OOPs 


## Objects
- **Definition**: An object is an instance of a class that contains both data (fields/attributes) and behavior (methods).
- **Explanation**: Objects represent real-world entities. They are created from classes and allow us to use the functionality defined in the class.

## Classes
- **Definition**: A class is a blueprint or template for creating objects.
- **Explanation**: It defines the structure (fields) and behavior (methods) that its objects will have.

## Methods
- **Definition**: A method is a block of code inside a class that performs a specific task.
- **Explanation**: Methods define the behavior of objects. They can take parameters, perform operations, and return values.
