Functions Overview (Java + DSA)
🔹 What is a Function?
A function (also called a method in Java) is a block of code that performs a specific task.
It helps break down complex problems into smaller, reusable pieces.

🔹 Key Definitions
- Function Declaration: The way a function is defined, including its name, return type, and parameters.
Example:

- Return Type: Specifies the type of value a function will return.
Example: int, double, String, or void (no return).
- Parameters (Arguments): Inputs passed into a function to work with.
Example: (int a, int b) in the add function.

Types of Functions
- Built-in Functions
Predefined methods provided by Java libraries.
Example: Math.max(10, 20)
- User-defined Functions
Functions created by the programmer to solve specific problems.
Example: int factorial(int n)
