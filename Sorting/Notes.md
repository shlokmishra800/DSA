### **Sorting Algorithms in Java**

**Sorting algorithms are techniques used to arrange elements of an array in a specific order (ascending or descending) to make data easier to search and process efficiently.**

---

### **Bubble Sort**

**Bubble Sort is a simple sorting algorithm that repeatedly compares adjacent elements and swaps them if they are in the wrong order. With each pass, the largest element moves (or “bubbles”) to its correct position at the end. This process continues until the array is fully sorted.**

---

### **Selection Sort**

**Selection Sort is a sorting algorithm that works by repeatedly finding the smallest element from the unsorted portion of the array and placing it at the beginning. It gradually builds a sorted portion of the array from left to right.**

---

### **Insertion Sort**

**Insertion Sort is a sorting algorithm that builds the sorted array one element at a time. It takes each element and inserts it into its correct position within the already sorted part of the array, similar to arranging cards in your hand.**
